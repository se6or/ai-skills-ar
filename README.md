# مهارات الذكاء الاصطناعي بالعربية — AI Skills AR

> مجموعة مهارات (*Skills*) مخصصة للذكاء الاصطناعي، تُحسّن أسلوب التواصل وتضبط طريقة الاستجابة وفق سياقات لغوية وثقافية محددة.

---

## ما هي المهارات (Skills)؟

المهارة هي ملف تعليمات منظّم يُحمَّل إلى نموذج الذكاء الاصطناعي لتعديل سلوكه في محادثة بعينها. تعمل هذه الملفات كـ"شخصيات" أو "أوضاع" قابلة للتبديل؛ إذ تُحدّد أسلوب اللغة، ومستوى الاختصار، وطريقة معالجة المحتوى التقني.

## طريقة الاستخدام

### مع Claude (claude.ai)

1. افتح المحادثة أو أنشئ مشروعاً (*Project*)
2. ارفع ملف `SKILL.md` المطلوب ضمن تعليمات المشروع أو ابدأ المحادثة بمحتواه
3. سيطبّق النموذج قواعد المهارة تلقائياً على كامل المحادثة

### مع واجهات برمجة التطبيقات (API)

```python
with open("baghdadi-pro/SKILL.md", "r", encoding="utf-8") as f:
    skill = f.read()

messages = [
    {"role": "user", "content": f"{skill}\n\n---\n\nسؤالك هنا"}
]
```

```markdown

version: 1.0
description: >
  وصف مختصر للمهارة وغرضها
---

## القواعد والتعليمات
...
```

## المساهمة

نرحّب بإضافة مهارات جديدة. لإضافة مهارة:

1. أنشئ مجلداً باسم المهارة
2. أضف ملف `SKILL.md` داخله وفق البنية الموضحة أعلاه
3. أرسل *Pull Request* مع وصف واضح لما تُقدّمه المهارة
